# Shinsei Village Landing Page

A visually immersive and responsive anime-inspired landing page built using **HTML5** and **CSS3**.

🔗 **Live Demo:** 

https://im-ayushgowda.github.io/Cohort-3.0/Assignment-08/

## ✨ Features

- Responsive full-screen hero section
- Glassmorphism navigation bar
- Custom Google Fonts
- Blurred fantasy background effect
- Character showcase image
- Styled CTA “Mint Now” button
- Vision section with curved transition layout
- Fully responsive design for tablets and mobile devices

## 🛠️ Technologies Used

  - Audiowide
