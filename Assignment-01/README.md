# ⚡ Pikachu - Pokemon Gen 1

A visually stunning Pokémon showcase card built with pure HTML and CSS, featuring Pikachu from Generation 1.

## 🔴 Live Demo

👉 [Click here to view the live site](https://im-ayushgowda.github.io/Cohort-3.0/Assignment-01/)

---

## 📸 Features

- ⚡ Split background layout (Yellow + Beige)
- 🎨 CSS-drawn Pokeball watermark
- 🖼️ Pikachu hero image with drop shadow
- 🔤 Google Fonts (Poppins)
- 📱 Clean and minimal UI
- 🔵 Pagination dots sidebar

---

## 🛠️ Built With

- HTML5
- CSS3
- [Poppins - Google Fonts](https://fonts.google.com/specimen/Poppins)
- [Remix Icons](https://remixicon.com/)
