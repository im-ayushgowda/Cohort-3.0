# Aurora Winter Drop

A futuristic winter fashion landing page built using pure HTML and CSS.  
Inspired by premium fashion brand interfaces and cyberpunk-inspired product showcases.

---

## 🚀 Live Demo

🌐 Live Website:

https://im-ayushgowda.github.io/Cohort-3.0/Assignment-03/



## ✨ Features

- Futuristic winter fashion UI
- Fully responsive layout
- Hero product showcase section
- Product cards with overlay effects
- Glassmorphism-inspired styling
- Gradient cyberpunk background
- Feature information strip
- Modern typography using Google Fonts
- Pure HTML & CSS only

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Flexbox
- CSS Gradients
