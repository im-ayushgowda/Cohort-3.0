# Deep°Style — Fashion Landing Page

A stylish fashion landing page built with pure HTML and CSS as part of **Cohort 3.0 — Assignment 04**.

## 🔗 Live Demo

https://im-ayushgowda.github.io/Deep---Style-Assignment-4/

---

## 📁 Project Structure

```
Assignment-04/
├── index.html       # Main HTML file
└── style.css        # All styles
```

---

## ✨ Features

- **Navbar** with animated hover underline on nav links and lord-icon animated icons
- **Hero section** with a bold headline and decorative flower emoji assets
- **Summer Collection** banner with text and image layout
- **Asymmetric Cards Grid** — a 5-column CSS Grid with overlapping fashion images
- **Footer** with a quote, scroll button, and blog post preview

---

## 📐 Layout Highlights

- Navbar uses `flexbox` with `gap` and centered alignment
- Cards section uses `CSS Grid` with `position: absolute` for the overlapping magazine-style layout
- Footer uses `flexbox` with `align-items: end` for the bottom-aligned three-column design
- Nav links use a `::after` pseudo-element for the smooth underline hover animation
