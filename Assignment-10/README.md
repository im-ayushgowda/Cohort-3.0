# MY-TASK — Todo List App

A simple, beginner-friendly Todo List web app built with HTML, CSS, and vanilla JavaScript.

🔗 **Live Demo:** https://im-ayushgowda.github.io/Cohort-3.0/Assignment-10/

---

## 📁 Project Structure

```
Assignment-10/
├── index.html       → Structure of the page
├── style.css        → Styling and layout
├── script.js        → All the logic (add, edit, delete)
└── README.md        → Project documentation
```

---

## ✨ Features

- ✅ Add a new task by typing and clicking ADD or pressing Enter
- ✅ Edit any task inline — click Edit, change text, press Enter or click Save
- ✅ Delete any task instantly
- ✅ Empty message shows when no tasks are present
- ✅ Auto-focus on input when page loads

## 📖 How It Works

### Adding a Task
1. Type your task in the input box
2. Click **ADD** or press **Enter**
3. Task appears in the list below

### Editing a Task
1. Click the **Edit** button on any task
2. The task text becomes an editable input
3. Make your changes
4. Press **Enter** or click **Save**

### Deleting a Task
1. Click the **Delete** button on any task
2. Task is removed instantly
3. If no tasks remain, the empty message appears again

## 🧠 Key JavaScript Concepts Used

- `document.querySelector` — selecting elements
- `addEventListener` — listening for clicks and keypresses
- `createElement` — dynamically creating HTML elements
- `appendChild` — adding elements to the page
- `replaceWith` — swapping elements (used in edit)
- `remove()` — deleting elements from the page
- `textContent` — getting and setting text
- `style.display` — showing and hiding elements
