[
[https://im-ayushgowda.github.io/lombok-wellness/](https://im-ayushgowda.github.io/lombok-wellness/)](https://im-ayushgowda.github.io/Cohort-3.0/Assignment-05/)

## 📌 Features

* Responsive layout
* Modern navigation bar
* Wellness-inspired design
* Clean typography and spacing
* Hero section with stylish visuals
* Custom CSS styling

## 🛠️ Technologies Used

* HTML5
* CSS3
* Remix Icons

## 💻 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/im-ayushgowda/lombok-wellness.git
```

2. Open the project folder:

```bash
cd lombok-wellness
```

3. Open `index.html` in your browser.
