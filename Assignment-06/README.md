# Gunaforycter Restaurant Menu

A premium restaurant and drinks showcase landing page built using pure HTML and CSS.  
Designed with an elegant luxury dining aesthetic featuring modern cards, chef specials, and responsive layouts.

---

## 🚀 Live Demo

🌐 Live Website:

https://im-ayushgowda.github.io/Cohort-3.0/Assignment-06/

## ✨ Features

- Elegant restaurant landing page
- Premium drink showcase cards
- Chef’s special featured section
- Modern responsive layout
- Luxury typography styling
- Hover-ready UI components
- Footer navigation with social icons
- Fully built using HTML & CSS

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Flexbox
- Responsive Design
