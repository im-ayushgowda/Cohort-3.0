# Explore Art Work

A modern digital art showcase landing page built using pure HTML and CSS.  
Designed with a futuristic glassmorphism-inspired UI featuring artwork cards, gradients, and responsive layouts.

---

## 🚀 Live Demo

https://im-ayushgowda.github.io/Cohort-3.0/Assignment-07/

## ✨ Features

- Modern art gallery interface
- Responsive card-based layout
- Glassmorphism UI design
- Gradient futuristic background
- Interactive artwork cards
- Stylish typography using Google Fonts
- Clean and minimal user interface
- Pure HTML & CSS only

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Flexbox
- CSS Gradients
- Responsive Design
